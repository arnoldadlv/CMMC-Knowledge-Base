# CMMC Level 2 Knowledge Base

An AI-consumable CMMC expert knowledge base built from 27 source files — 10 Discord community research articles and 17 Summit 7 YouTube transcript analyses. Organized into topics, decision frameworks, and reference materials for anyone preparing for CMMC Level 2 certification.

## How to Use

**In Obsidian:** Clone this repo and open it as a vault. All internal links (`[[wikilinks]]`) will resolve automatically. Start from `_MOC.md` to navigate the full knowledge base.

**On GitHub:** Browse the folders below or start from [_MOC.md](_MOC.md) as the main index.
